## What is Vagrant?

- Vagrant is a tool for building and managing virtual machine environments in a single workflow.
- With an easy-to-use workflow and focus on automation, Vagrant lowers:
  - Development environment setup time
  - Increases production parity
  - And makes the "works on my machine" excuse a relic of the past.

## What is Oracle Virtual Box?

- VirtualBox is a powerful x86 and AMD64/Intel64 virtualization product
- It holds the virtual machines or boxes that we can use to test products in a controlled manner.
- It is used in conjunction with vagrant

### How to install and initialise Vagrant

- Installing vagrant is simple. Head over to the website and download the installer. Follow the instructions and viola!
- Using git bash:
  - Head over to your repository of choice and git init
  - Type in: vagrant init <name of the box or operating system you want to use> eg: <ubuntu/xenial64>
  - Vagrant init initialises vagrant and creates a vagrant file in the folder of choice where you can make changes to the operating system you chose.
